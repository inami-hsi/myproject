## Animation

### Hover

- **Duration**: 150ms
- **Easing**: ease-out
- **Properties**: opacity, transform(scale)

```css
.button-hover {
  transition: opacity 150ms ease-out, transform 150ms ease-out;
}

.button-hover:hover {
  opacity: 0.9;
  transform: scale(1.02);
}
```

### Active (Press)

- **Transform**: `scale(0.98)` on press
- **Duration**: 100ms

### Focus

- **Ring**: `ring-2 ring-accent/20 ring-offset-2`
- **Outline**: none (replaced by ring)
- **Visibility**: 明確に視認可能

```css
.button:focus-visible {
  outline: none;
  box-shadow: 0 0 0 2px var(--color-background), 0 0 0 4px rgba(224, 122, 95, 0.2);
}
```

### Disabled

- **Opacity**: 0.5
- **Pointer Events**: none
- **Cursor**: not-allowed

### Loading

- Spinner icon (Loader2 from Lucide, `animate-spin`)
- テキストを維持し、左側にスピナーを表示
- `pointer-events: none` でクリック不可

## Accessibility

### Requirements

| Item | Requirement |
|------|-------------|
| **aria-label** | アイコンのみのボタン（`icon` サイズ）には必須 |
| **aria-disabled** | `disabled` 状態では `aria-disabled="true"` を付与 |
| **aria-busy** | ローディング中は `aria-busy="true"` を付与 |
| **role** | `<button>` 要素を使用。`<a>` を使う場合は `role="button"` |
| **keyboard** | Enter / Space で押下。フォーカス可能 |
| **focus indicator** | `focus-visible` でリングを表示。マウスクリック時は非表示 |

### Contrast Check

| Combination | Contrast Ratio | WCAG |
|-------------|---------------|------|
| White on Accent (`#e07a5f`) | 3.1:1 | PASS (large text / UI component) |
| White on Danger (`#c44e4e`) | 4.1:1 | PASS (AA) |
| Text (`#2d2d2d`) on Secondary (`#f8f8f2`) | 11.8:1 | PASS (AA) |

**Note**: `default` variant (white on accent) のコントラスト比は 3.1:1 であり、WCAG AA のUIコンポーネント基準 (3:1) を満たす。ただし、ボタン内テキストは 14px Bold 以上であること。

## Implementation

### cva Definition

```tsx
import { cva, type VariantProps } from "class-variance-authority";

const buttonVariants = cva(
  [
    "inline-flex items-center justify-center gap-2",
    "rounded-lg",
    "font-heading font-medium",
    "transition-all duration-150 ease-out",
    "focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-accent/20 focus-visible:ring-offset-2",
    "active:scale-[0.98]",
    "disabled:pointer-events-none disabled:opacity-50",
    "whitespace-nowrap",
  ],
  {
    variants: {
      variant: {
        default:
          "bg-accent text-white hover:bg-accent/90",
        destructive:
          "bg-danger text-white hover:bg-danger/90",
        outline:
          "border border-border bg-transparent hover:bg-secondary text-text",
        secondary:
          "bg-secondary text-text hover:bg-secondary/80",
        ghost:
          "hover:bg-secondary text-text",
        link:
          "text-accent underline-offset-4 hover:underline",
      },
      size: {
        default: "h-10 px-4 text-sm",
        sm: "h-8 px-3 text-sm",
        lg: "h-12 px-6 text-base",
        icon: "h-10 w-10",
      },
    },
    defaultVariants: {
      variant: "default",
      size: "default",
    },
  }
);
```

### Component

```tsx
import * as React from "react";
import { Slot } from "@radix-ui/react-slot";
import { cn } from "@/lib/utils";
import { Loader2 } from "lucide-react";

export interface ButtonProps
  extends React.ButtonHTMLAttributes<HTMLButtonElement>,
    VariantProps<typeof buttonVariants> {
  asChild?: boolean;
  loading?: boolean;
}

const Button = React.forwardRef<HTMLButtonElement, ButtonProps>(
  (
    { className, variant, size, asChild = false, loading, children, disabled, ...props },
    ref
  ) => {
    const Comp = asChild ? Slot : "button";

    return (
      <Comp
        className={cn(buttonVariants({ variant, size, className }))}
        ref={ref}
        disabled={disabled || loading}
        aria-busy={loading || undefined}
        aria-disabled={disabled || loading || undefined}
        {...props}
      >
        {loading && <Loader2 className="w-4 h-4 animate-spin" />}
        {children}
      </Comp>
    );
  }
);

Button.displayName = "Button";

export { Button, buttonVariants };
```
