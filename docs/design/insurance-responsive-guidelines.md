## 概要

このドキュメントは、モバイル・タブレット・デスクトップ全デバイスでの**一貫性のある使用体験**を実現するためのレスポンシブデザイン仕様です。

---

## 1. ブレークポイント戦略

### 1.1 ブレークポイント定義

```
sm:  375px   スマートフォン（最小値、iPhone SE相当）
md:  768px   タブレット（iPad, Android タブレット）
lg:  1024px  デスクトップ（一般的なPC）
xl:  1280px  デスクトップ大（高解像度PC）
```

### 1.2 モバイルファースト戦略

**基本方針:**
```
1. モバイル (375px) から設計を開始
2. md (768px) で機能追加 / レイアウト拡張
3. lg (1024px) で最終デスクトップレイアウト
4. xl (1280px) で大画面最適化
```

**実装例 (CSS):**
```css
/* モバイルファースト（基本） */
.button {
  padding: 12px 24px;
  font-size: 14px;
}

/* タブレット対応 */
@media (min-width: 768px) {
  .button {
    font-size: 16px;
  }
}

/* デスクトップ対応 */
@media (min-width: 1024px) {
  .button {
    padding: 12px 32px;
  }
}
```

## 10. ビューポート設定

**HTML meta タグ:**
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=5.0, user-scalable=yes">
```

**説明:**
- `width=device-width`: デバイス幅に合わせる
- `initial-scale=1.0`: 初期表示ズーム100%
- `maximum-scale=5.0`: ユーザーズーム許容
- `user-scalable=yes`: ユーザー拡大縮小可能
