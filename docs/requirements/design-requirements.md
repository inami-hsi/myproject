## 1. 美的方向性（Aesthetic Direction）

### 1.1 トーン

- [x] **calm-functional** - 穏やかで機能的。集中力を高めるクリーンなデザイン

タスク管理ツールは毎日長時間使う道具である。派手さよりも「邪魔をしない」ことを最優先とし、ユーザーがタスクの内容そのものに集中できる静かなインターフェースを目指す。

### 1.2 ムード

- [x] professional（プロフェッショナル）

穏やかで集中力を高める空間。視覚ノイズを最小化し、タスクの内容に注意を向けさせる。色数を抑え、余白を十分に確保することで「呼吸感」のあるUIを実現する。

### 1.3 参照サイト

| サイト名 | URL | 参考ポイント |
|---------|-----|-------------|
| Linear | linear.app | ミニマルUI、キーボードショートカット中心の操作、余白設計 |
| Notion | notion.so | クリーンなタイポグラフィ、柔軟なレイアウト、コンテンツファースト |
| Height | height.app | ガントチャートUI、タスク管理UX、穏やかな色使い |

### 1.4 禁止パターン（AI Slop回避）

以下のデザインパターンは「AI生成の典型」として禁止する。

| 禁止パターン | 理由 |
|-------------|------|
| Inter, Roboto, Arial等のジェネリックフォント | AIが最も多く選択するフォント。個性がない |
| Space Grotesk | AI生成アプリの典型フォント |
| 紫グラデーション on 白背景 | AI生成UIの典型配色 |
| 予測可能なレイアウトパターン | Hero + 3カラムグリッド等の定型パターン |
| 過剰なドロップシャドウ | カード全てにshadow-xlを適用する等 |
| グロー効果 | ネオンのような発光効果 |
| Cookie-cutter design | テンプレート感のある没個性デザイン |

### 1.5 差別化ポイント

穏やかな色調と十分な余白で「呼吸感」のあるガントチャート。

従来のガントチャートツール（MS Project, Asana, Monday.com等）が持つ「情報密度の高さ = 使いにくさ」の問題を排除する。タスクの流れを直感的に理解でき、開いた瞬間に「今何をすべきか」が分かるデザイン。

**一つ覚えてもらうなら**: 「静かで心地よいガントチャート」

- ガントバーの角丸: 6px (`rounded-md`)
- バー高さ: 28px（デスクトップ）、36px（タッチデバイス）
- バー間の余白: 8px

### 6.2 進捗率表示

- ガントバー内に進捗率をインライン表示（バー幅が80px以上の場合）
- フォント: DM Sans 500, 12px
- 色: バー色に対するコントラスト確保（白またはダークテキスト）

### 6.3 依存関係の矢印

- デフォルト: `#9ca3af` (控えめグレー), 1px実線

- 色: `#c44e4e` (Danger)

### 6.5 マイルストーン

- 形状: ダイヤモンド（45度回転した正方形）
- サイズ: 16px x 16px

## 7. モーションガイドライン

### 7.1 基本原則

- 全てのインタラクションアニメーションは **200ms以下**
- **Compositor props のみ**使用: `transform`, `opacity`
- レイアウトプロパティ（`width`, `height`, `top`, `left`）のアニメーションは禁止
- `prefers-reduced-motion` を尊重

### 7.2 アニメーション定義

| 操作 | duration | easing | プロパティ |
|------|----------|--------|----------|
| ボタンホバー | 150ms | ease-out | opacity, transform(scale) |
| カード展開 | 200ms | ease-out | opacity, transform |
| モーダル表示 | 200ms | ease-out | opacity, transform(translateY) |
| サイドバー開閉 | 200ms | ease-in-out | transform(translateX) |
| ガントバースケール切替 | 200ms | ease-out | transform(scaleX) |
| ドロップダウン | 150ms | ease-out | opacity, transform(translateY) |
| トースト通知 | 200ms | ease-out | opacity, transform(translateX) |
| ページ遷移 | 150ms | ease-in-out | opacity |

### 7.3 禁止事項

- `will-change` の常時適用（必要な時のみ）
- `backdrop-filter: blur()` の過度な使用（パフォーマンス影響）
- 300ms超のアニメーション
- bounce, elastic等の過度なイージング
