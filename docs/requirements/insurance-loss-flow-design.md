## 2. 自動車保険フロー

### 2.1 ヒアリング質問（7～8問）

#### 質問1: 年齢条件
```
【質問】
年齢条件をお選びください。

【選択肢】
A. 18～20歳
B. 21～25歳
C. 26～34歳
D. 60歳以上

【推奨に影響する軸】
- 事故リスク評価
- サービス内容の優先度（若年層→保険料重視、シニア→サービス重視）
- 特約の適合性
```

#### 質問2: 運転用途
```
【質問】
あなたの運転パターンに最も近いものを選んでください。

【選択肢】
A. 業務使用
B. 通勤・通学使用
C. 日常・レジャー使用

【推奨に影響する軸】
- 事故リスク（使用目的による運転条件の違い）
- 保険料（用途別の料率区分）
- 補償内容（使用目的別のカスタマイズ）
```

#### 質問3: 家族の運転状況
```
【質問】
実際に車を運転する方は何人ですか？

【選択肢】
A. 本人のみ（限定特約の対象）
B. 本人＋配偶者（配偶者特約の対象）
C. 本人＋配偶者＋成人の子ども（複数運転者管理）
D. 本人＋同居の親や複数の家族（多数運転者管理）

【推奨に影響する軸】
- 複数運転者対応の充実度
- 個別管理vs一括管理の提案
- 年齢条件の設定柔軟性
```

#### 質問4: 過去の事故経歴
```
【質問】
過去3年間の事故経歴をお聞きします。

【選択肢】
A. 事故なし（優良ドライバー）
B. 物損事故のみ（1件）
C. 物損事故複数件（2件以上）
D. 人身事故経験あり（等級ダウンの可能性あり）

【推奨に影響する軸】
- 等級下げの制度対応（割引制度との相性）
- 事故サポートの充実度
- 特約の重視度（事故対応サービス）
```

#### 質問5: 利用する車種・用途
```
【質問】
加入する車はどのような車ですか？

【選択肢】
A. 普通乗用車（セダン・SUV）
B. 軽自動車
C. 商用車・トラック（事業中心利用）
D. 高額車両（500万円以上含む）

【推奨に影響する軸】
- 保険会社の受け入れ可否（商用車対応など）
- 保険料（車種による差）
- 特約オプション
```

#### 質問6: 補償内容の優先度（3つまで選択）
```
【質問】
あなたが重視する補償内容を、優先順位で選んでください。

【選択肢（複選、上位3つで点数付け）】
① 事故対応の丁寧さ・スピード（対応力重視）
② 保険料の安さ（コスト重視）
③ 弁護士費用特約の充実度
④ ロードサービス・24時間サポート（安心重視）
⑤ デジタル対応・オンライン手続き（利便性重視）
⑥ レンタカー・代車費用特約

【推奨に影響する軸】
- この質問が最もスコアリング上重要
- 各保険会社の差別化ポイントと直結
```

#### 質問7: 加入形式・相談方式
```
【質問】
保険加入時の相談方式をどのようにされたいですか？

【選択肢】
A. 対面で代理店スタッフに相談したい（信頼性重視）
B. オンライン（チャット・Zoom）で相談したい（手軽さ重視）
C. Webで自分で見積・加入したい（独立性重視）
D. 特にこだわらない

【推奨に影響する軸】
- 代理店ネットワークの充実度
- デジタルプラットフォームの充実度
```

#### 質問8: 追加サービスへの関心
```
【質問】
以下の付加サービスにご関心はありますか？

【選択肢（複選）】
☐ 健康診断・運転能力診断（無料受診）
☐ 自動運転支援アプリ（安全運転でポイント還元）
☐ 走行データ分析（リアルタイム安全アドバイス）
☐ 特になし

【推奨に影響する軸】
- 付加価値サービス充実度
```

## 3. 火災保険フロー

### 3.1 ヒアリング質問（7～8問）

#### 質問1: 対象物の種別
```
【質問】
加入する建物の種別をお選びください。

【選択肢】
A. 戸建て（一戸建て住宅・持家）
B. マンション（分譲マンション・持家）
C. 賃貸住宅（アパート・賃貸マンション）
D. 併用住宅（住宅＋店舗等）

【推奨に影響する軸】
- 火災保険の種別（世帯向け/マンション専用/賃貸/併用）
- 各保険会社の対応範囲
```

#### 質問2: 建物築年数
```
【質問】
建物の築年数はどのくらいですか？

【選択肢】
A. 5年未満（新築・新しい）
B. 5～15年
C. 15～30年
D. 30年以上（老朽化懸念）

【推奨に影響する軸】
- 災害リスク評価（古い建物→リスク上昇）
- 保険料（築年数で大きく変動）
```

#### 質問3: 保障対象（建物/家財）
```
【質問】
保険で守りたい対象物をお選びください。

【選択肢】
A. 建物のみ（家の躯体・構造体）
B. 家財のみ（家具・家電・衣類等）
C. 建物＋家財（両方）
D. 什器備品や商品も含む

【推奨に影響する軸】
- 保険料（C > A,B）
- 補償内容の提案内容
```

#### 質問4: 補償してほしい災害
```
【質問】
補償してほしい災害をお選びください（複選、上位2つ）。

【選択肢（複選）】
☐ 豪雨・洪水・内水氾濫（床上浸水等）
☐ 雹（ひょう）による被害（屋根・窓被害）
☐ 雪害（積雪による重み）
☐ 土砂崩れ
☐ 地震（別途地震保険が必要）
☐ 台風（高風）
☐ 盗難

【推奨に影響する軸】
- 特約の推奨内容
- 保険料設定（リスク地域の割増等）
```

#### 質問5: 建物構造・防火対策
```
【質問】
建物の構造と防火対策をお答えください。

【選択肢】
A. 木造・防火性低い
B. 木造・防火対策あり（火災警報器等）
C. 鉄骨造・防火性中程度
D. 鉄筋コンクリート造・防火性高い

【推奨に影響する軸】
- 火災リスク評価
- 保険料等級（防火性→割引率が上がる）
```

#### 質問6: 補償内容の優先度（3つまで選択）
```
【質問】
あなたが重視する補償をお選びください（上位3つ）。

【選択肢（複選、優先順位付け）】
① 基本補償を手厚く（火災・破裂爆発・落雷）
② 水災補償を厚く（豪雨・洪水対応）
③ 盗難補償を追加したい
④ 破損・汚損補償を追加したい
⑤ 保険料の安さ（基本補償に限定）
⑥ 特約の豊富さ（カスタマイズ重視）
⑦ 地震補償を厚くしたい（東京海上日動の超保険：地震危険100％）

【推奨に影響する軸】
- 補償内容の手厚さを求めるか否か
- 特約の豊富性
- 保険料との相関
```

#### 質問7: 長期契約への関心

【質問】
火災保険の契約期間をどのようにお考えですか？

## 8. 実装上の技術要件

### 8.1 フロントエンド

- **フレームワーク**: Next.js 14 (App Router)
- **スタイリング**: Tailwind CSS
- **UI コンポーネント**: shadcn/ui
- **状態管理**: Zustand（ユーザー回答の管理）
- **フォーム**: React Hook Form
- **チャート・図表**: Recharts（比較表など）

### 8.2 バックエンド

- **推奨ロジック**: Node.js で実装
- **スコアリング**: JSON-based scoring matrix
- **見積API**: 各保険会社のAPI連携（可能な範囲）

### 8.3 ストレージ

- ユーザーの回答履歴: LocalStorage + オプションでDB保存
- 保険会社マスタデータ: JSON ファイル（初版）
