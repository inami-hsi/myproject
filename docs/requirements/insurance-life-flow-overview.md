## 1. 生命保険の全体構成

### 1.1 対象保険商品（8種類）

```
【死亡保険】
├─ 定期保険（10年・20年など期間限定）
├─ 終身保険（一生涯の保障）
├─ 収入保障保険（月額給付型）
└─ 就業不能保険（働けない時のサポート）

【医療・疾病保険】
├─ 医療保険（入院・手術補償）
├─ がん保険（がん特化型保障）
└─ 介護保険（介護状態時の補償）

【資産形成・年金】
├─ 個人年金保険（老後資金確保）
├─ 変額保険（投資型、リスク・リターンあり）
├─ 一時払い終身保険（相続対策、保障+相続増額）
└─ 学資保険（子どもの教育資金）

【特殊対応型】
├─ 引き受け基準緩和型医療保険（既往症OK）
└─ 引き受け基準緩和型死亡保険（健康診断結果OK）
```

### 1.2 顧客セグメント別の推奨パターン（案）

| セグメント | 優先ニーズ | 推奨保険 |
|---------|---------|--------|
| **20-30代独身** | 備え+貯蓄並立 | 定期保険 + 医療保険 |
| **30-40代子どもあり** | 遺族保障 | 定期保険（終身まで）+ 医療保険 + がん保険 |
| **45-55代更新期** | 保障見直し + 相続対策 | 終身保険 + 個人年金 + がん保険 |
| **60代以上** | 医療保障 + 遺産分割 | 医療保険 + 一時払い終身保険 |
| **健康上の懸念** | 入りやすい保障 | 引き受け基準緩和型 |

## 2. 生命保険ヒアリング設計の基本骨子

### 2.1 多次元的ヒアリングアプローチ

生命保険は損害保険と異なり、「**ライフプラン**」と密結する。以下の軸で段階的に質問を構成：

```
【レイヤー1】基本情報（1-2問）
  ↓
【レイヤー2】人生設計・ライフステージ（2-3問）
  ↓
【レイヤー3】ニーズの優先度（3-4問）
  ↓
【レイヤー4】健康状況・引き受け条件（1問）
  ↓
【推奨結果】
```

### 2.2 各保険種目ごとのヒアリング構成（概要）

#### A. 死亡保険（定期保険）

**質問数**: 7～9問

**主要質問軸**:
1. 年齢層・性別
2. 家族構成（扶養家族人数）
3. 必要な遺族保障額（教育費・生活費を計算）
4. 保障期間（子どもの成人まで/定年まで）
5. 現在の貯蓄額（補保必要額を算出）
6. 職業分類（リスク評価）
7. 優先度（保険料重視 vs 保障充実）
8. 健康診断結果（基本VS簡易）
9. 付加サービス期待度

**推奨ポイント**:
- 必要保障額の自動計算ロジック（家族構成→ライフイベント費用算出）
- 各保険会社の「定期保険」の商品差別化は微小 → ユーザー期待値は「保険料安さ」「デジタル対応」に集約
- 推奨理由は「あなたの家族構成で月額○○円で△△円の保障を実現できる」と**具体数字推し**

#### B. 終身保険

**質問数**: 6～7問

**主要質問軸**:
1. 人生設計（相続対策の関心度）
2. 資産形成志向か保障志向か
3. 返戻率への関心度
4. 相続税対策への関心度
5. 月々の支払可能額
6. 長期保有の覚悟度

**推奨ポイント**:
- 「相続対策」「資産形成」の関心に大きく左右
- 三井住友海上プライマリー生命 vs アフラック vs 日本生命の返戻率比較が鮮明
- 「貯蓄性」重視層には個人年金保険の検討も提案

#### C. 医療保険

**質問数**: 8～9問（最もボリューム多い）

**主要質問軸**:
1. 年齢層（保険料が年齢で大きく変動）
2. 職業分類（高リスク職か否か）
3. 入院給付日額（5千/1万/2万円）
4. 通院補償への関心度
5. 手術特約・先進医療特約への関心
6. 健康診断結果（基本vs簡易）
7. 既往症・既検査（基本型 vs 緩和型の判定）
8. サービス面での優先（保険料 vs 給付金手厚さ）
9. 更新型か終身型か

**推奨ポイント**:
- 「既往症あり」「健診所見あり」の場合は**緩和型を即推奨**
- 若年層には「安さ」（ネオファースト・損保系） vs 中年層には「手厚さ」（日本生命・東京海上）で分化
- 診断給付金の「ワンタイムvs継続」の関心が分岐点

#### D. がん保険

**質問数**: 7～8問

**主要質問軸**:
1. がん保険加入の背景（家族歴/一般対策）
2. 診断給付金vs継続給付金の選好
3. 先進医療特約への関心
4. 上皮内新生物への関心
5. 既往症・がん検査履歴
6. 月々の支払可能額
7. アフラック（がん保険業界先駆者） vs 他社の評価

**推奨ポイント**:
- **アフラック** vs **三井住友海上あいおい生命** vs **東京海上あんしん生命** の3社で大半が網羅される
- 「診断給付金一括」派 → アフラック推奨（ミライト等で充実）
- 「先進医療」重視派 → 東京海上推奨

#### E. 個人年金保険

**質問数**: 6～7問

**主要質問軸**:
1. 老後資金の必要額（年金生活シミュレーション）
2. 運用リスク許容度（保証型 vs 変額）
3. 受け取り方（確定年金 vs 終身年金）
4. 返戻率への関心度
5. 相続機能への関心
6. 月々の支払可能額

**推奨ポイント**:
- **三井住友海上プライマリー生命**（返戻率高い）vs **日本生命**（手厚いサポート）で分化
- 「貯蓄性」重視 → プライマリー
- 「安定性」「サポート」重視 → 日本生命

#### F. 変額保険

**質問数**: 8～9問

**主要質問軸**:
1. 投資経験有無
2. 運用知識の深さ
3. リスク許容度スペクトラム（積極→保守）
4. 長期保有の覚悟
5. 保障と運用のバランス希望
6. 月々の支払可能額

**推奨ポイント**:
- 「投資経験あり」「リスク許容度高い」に限定される
- アクサ生命、三井住友海上プライマリーが主流
- 初心者には「終身保険」「個人年金」推奨（変額は不推奨）

#### G. 引き受け基準緩和型保険

**質問数**: 5～6問

**主要質問軸**:
1. 既往症の種類・治療状況
2. 通院・入院履歴
3. 健康診断での指摘事項
4. 基本型で加入可能か確認フロー
5. 緩和型での保険料・保障内容への理解

**推奨ポイント**:
- **判定ロジック**: 既往症タイプごとに対応保険会社がほぼ決まる
  - 例: 糖尿病 → アクサ生命（対応実績多い）
  - 例: がん既往 → アフラック（対応充実）
- 「基本型で加入できるか」を**予め自動判定**することで、ユーザーストレス軽減

## 3. 生命保険の推奨ロジックの方針（概要）

### 3.1 推奨スコアリングの軸（案）

生命保険は損害保険以上に「**ユーザーの人生設計**」に左右されるため、以下の軸を重視：

| 軸 | 说明 | 影響度 |
|----|------|--------|
| **保障額の適正化** | 回答から必要保障額を自動計算 → その額を実現できるか | ★★★★★ |
| **人生ステージマッチング** | ユーザーの年齢・家族構成から最適商品を提案 | ★★★★★ |
| **保険料負担感** | 月額保険料とニーズのバランス | ★★★★☆ |
| **返戻率・貯蓄性** | 個人年金・終身保険での重要指標 | ★★★★☆ |
| **健康告知対応** | 既往症・健康診断結果への対応範囲 | ★★★★☆ |
| **付加サービス** | 健康相談サービス、医療情報提供など | ★★★☆☆ |
| **ブランド信頼度** | 保険会社の規模・歴史・格付け | ★★☆☆☆ |

### 3.2 会社別の推奨パターン（概要）

#### パターン①: 「若年家計ニーズ型」（20-30代）

**最優先**: 三井住友海上あいおい生命
- 収入保障保険で「月額給付」で生活費カバー
- 医療保険による基本補償
- 価格競争力あり

**次点**: アフラック（医療+がん）

---

#### パターン②: 「子育て期間型」（30-45代）

**最優先**: 日本生命
- 定期保険で大型保障（両親の教育費・住宅費カバー）
- 医療保険で入院対応
- サポート充実（相談体制）

**次点**: 東京海上日動あんしん生命

---

#### パターン③: 「資産形成志向型」（40-55代）

**最優先**: 三井住友海上プライマリー生命
- 個人年金保険で返戻率重視
- 変額保険での投資性
- 相続対策機能

**次点**: アクサ生命

---

#### パターン④: 「既往症・健康懸念型」

**最優先**: アクサ生命 OR ネオファースト生命
- 緩和型保険の対応実績多い
- 加入可能性が高い

**次点**: アフラック緩和型

### 4.2 「フリーフロー」の自動推奨ロジック（案）

ユーザーが「保険種目わからない」場合、**ライフステージクイズ**で自動推奨：

```
Q. あなたの状況は？
A. 独身・まだ親の扶養中 → 「医療保険」を優先提案
B. 結婚したばかり → 「定期保険」「医療保険」を順序で提案
C. 子どもがいる → 「定期保険」「医療保険」「学資保険相談」を提案
D. 子どもが独立 → 「個人年金」「終身保険」「がん保険」を提案
E. 60代以上 → 「個人年金」「医療保険」「一時払い終身」を順序で提案
F. 健康に不安がある → 「緩和型医療」「緩和型死亡」を優先提案
```

## 6. 今後の詳細設計フェーズ（損保実装後）

### 6.1 詳細化する項目

1. **各保険種目ごと7～9問の詳細ヒアリング文案作成**
2. **スコアリング表の詳細化**（具体数値・返戻率・保険料目安）
3. **推奨理由テキストの大量作成**（パターン×会社 = 50+パターン分）
4. **ユーザーシナリオ集の作成**（シミュレーション例：「30才、子2人の場合」など）
5. **ワイヤーフレーム詳細化**（質問ページレイアウト、結果表示パターン）

### 6.2 詳細化のトリガー

- **タイミング**: 損害保険フロー実装が基本完成した時点
- **リソース**: 同じチームで、知見を引き継いでから詳細設計開始
- **期間**: 詳細設計1～2週間 + 実装3～4週間
