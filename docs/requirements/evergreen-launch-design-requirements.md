## 1. 美的方向性（Aesthetic Direction）

### 1.1 トーン

- [x] **luxury-refined** - ラグジュアリー、洗練、高品質タイポグラフィ

エバーグリーンローンチは「限定感」と「プレミアム感」が重要。
高額商品の販売を前提としたデザインで、信頼と価値を視覚的に伝える。

### 1.2 ムード

- [x] **professional**（プロフェッショナル）
- [x] **elegant**（エレガント）

ビジネス向けの信頼感と、感情に訴える洗練されたデザインの融合。

### 1.4 禁止パターン（AI Slop回避）

以下は**使用禁止**:

- [x] Inter, Roboto, Arial等のジェネリックフォント
- [x] 紫グラデーション on 白背景
- [x] 予測可能なレイアウトパターン
- [x] 過剰なドロップシャドウ
- [x] Space Grotesk（AI生成の典型）
- [x] cookie-cutter design
- [x] ストックフォト感のあるヒーローイメージ


- スクロールに応じたフェードインアニメーション

## 6. アニメーション方針

### 6.1 基本ルール

| ルール | 詳細 |
|--------|------|
| 最大デュレーション | 200ms（インタラクション系） |
| トランジションプロパティ | transform, opacity のみ（compositor props） |
| イージング | ease-out（出現）、ease-in（消失） |

### 6.2 使用シーン

| シーン | アニメーション | ライブラリ |
|--------|------------|-----------|
| カウントダウン数字変更 | フリップ / スライドダウン | motion/react |
| ページ間トランジション | フェードイン | motion/react |
| CTAボタン出現 | スケールアップ + フェードイン | motion/react |
| スクロールセクション | IntersectionObserver + フェードアップ | CSS |
| 動画終了→オファー | スライドアップ | motion/react |
| 登録完了 | チェックマーク + コンフェッティ | CSS |

### 6.3 禁止アニメーション

- layout プロパティ（width, height, top, left）のアニメーション
- 3秒以上の連続アニメーション
- 自動再生されるアニメーション（prefers-reduced-motion 考慮）

## 8. 追加要件

### 8.1 カウントダウンタイマー

- セッション開始までの残り時間をリアルタイム表示
- 日・時・分・秒の4ユニット
- tabular-nums でフォントが揺れないように固定幅
- 0になったら自動的に動画プレーヤーに切り替え

### 8.2 動画プレーヤー

- カスタムコントロール（ブランドカラー適用）
- プログレスバーはアクセントカラー
- フルスクリーン対応
- PiP（ピクチャーインピクチャー）対応

### 8.3 メールデザイン

- react-email でHTML/テキスト両対応
- ブランドカラー統一
- モバイルレスポンシブ
- CTAボタンは大きく、タップしやすく
