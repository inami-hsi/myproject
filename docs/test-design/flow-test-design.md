## 1. 概要

### 1.1 目的

ユーザーが実際に行う一連の操作フロー（ユーザージャーニー）を通しで検証する。個別のE2Eテストが「機能単位」の検証であるのに対し、フローテストは「目的単位」の検証を行う。ユーザーが具体的なゴールを達成するまでの一連の操作が、途切れることなく完了できることを保証する。

### 1.2 E2Eテストとの違い

| 観点 | E2Eテスト | フローテスト |
|------|---------|-----------|
| 粒度 | 機能単位（1画面、1操作） | ジャーニー単位（複数画面横断） |
| 前提 | 各テスト独立 | 前のステップの結果が次の前提 |
| 目的 | 「機能が動く」ことの検証 | 「ユーザーが目的を達成できる」ことの検証 |
| データ | テスト固有のシードデータ | フロー内で動的に作成されたデータ |
| 所要時間 | 短い（数秒〜数十秒） | 長い（数分） |

### 1.3 テストフレームワーク

| 項目 | 技術 |
|------|------|
| E2Eフレームワーク | Playwright |
| テスト環境 | Docker Compose (Next.js + PostgreSQL) |
| データ管理 | フロー内で動的に作成・検証 |
| 録画 | Playwright video recording (全フロー) |

### 1.4 テストファイル配置

```
tests/
  e2e/
    flows/
      flow-001-new-user-onboarding.spec.ts
      flow-002-daily-usage.spec.ts
      flow-003-project-management.spec.ts
    helpers/
      flowState.ts       # フロー間状態管理
```

## 5. エクスポートファイル検証

### 5.1 CSVエクスポート検証

```typescript
// テスト内での検証例
test("CSVエクスポートの内容が正しい", async ({ page }) => {
  const [download] = await Promise.all([
    page.waitForEvent("download"),
    page.click('[data-testid="export-csv-btn"]'),
  ]);

  const content = await download.suggestedFilename();
  expect(content).toMatch(/\.csv$/);

  const csv = await download.path();
  const text = await fs.readFile(csv!, "utf-8");
  const lines = text.split("\n").filter(Boolean);

  // ヘッダー行
  expect(lines[0]).toContain("title");
  expect(lines[0]).toContain("status");
  expect(lines[0]).toContain("priority");

  // データ行数
  expect(lines.length).toBeGreaterThanOrEqual(2); // ヘッダー + 1行以上
});
```

### 5.2 JSONエクスポート検証

```typescript
test("JSONエクスポートの内容が正しい", async ({ page }) => {
  const [download] = await Promise.all([
    page.waitForEvent("download"),
    page.click('[data-testid="export-json-btn"]'),
  ]);

  const json = await download.path();
  const text = await fs.readFile(json!, "utf-8");
  const data = JSON.parse(text);

  expect(Array.isArray(data)).toBe(true);
  expect(data.length).toBeGreaterThan(0);
  expect(data[0]).toHaveProperty("title");
  expect(data[0]).toHaveProperty("status");
  expect(data[0]).toHaveProperty("priority");
});
```

  const project = await page.request.post("/api/projects", {

    { title: "デザインモックアップ作成", status: "TODO", priority: "HIGH", progress: 0 },

  ];

  for (const task of tasks) {
    await page.request.post("/api/tasks", {
      data: { ...task, projectId },
    });
  }
