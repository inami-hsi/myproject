### アニメーション基準

| アニメーション種 | タイミング | イージング | 用途 |
|-------------------|----------|----------|------|
| **フェードイン** | 0.5s～1s | ease-out | 要素の出現 |
| **フェードアウト** | 0.5s～1s | ease-in | 要素の消滅 |
| **スライドイン（右）** | 0.8s | cubic-bezier(0.34, 1.56, 0.64, 1) | 左から右への導入 |
| **スライドイン（下）** | 0.8s | cubic-bezier(0.34, 1.56, 0.64, 1) | 上から下への導入 |
| **スケール** | 1s | ease-out | 拡大・強調 |
| **回転** | 1.5s～2s | linear | ローディング、循環 |

**禁止**: 紫グラデーション、70fps以上のアニメーション、3秒以上の単一アニメーション

| パターン | 禁止理由 | 代替案 |
|---------|---------|--------|
| **Inter フォント** | AI生成コンテンツ（AI slop） | Poppins + Lora を使用 |
| **Roboto フォント** | AI生成コンテンツ | Poppins + Lora を使用 |
| **Arial フォント** | Web標準的で個性がない | Poppins + Lora を使用 |

| **過度なドロップシャドウ** | 安っぽく見える | subtle shadow (0 1px 2px rgba) 程度 |
| **グロー効果** | Weblyなα感 | outline または border に変更 |
| **予測可能なレイアウト** | テンプレート感が強い | オフグリッド配置 + 非対称デザイン |
| **4秒以上の単一アニメーション** | テンポが悪い、視聴が進まない | 複数のマイクロアニメーション組み合わせ |

### 推奨パターン

| 要素 | 推奨実装 |
|------|---------|
| **色の使い分け** | Primary（情報・リード）、Success（肯定・完了）、Danger（警告・重要） |
| **テキスト配置** | 複数段落は左揃え、タイトルはセンター |
| **グラフ視覚** | 棒グラフは下から上へ伸長、折れ線は左から右へペイント |
| **ボックス枠線** | 2～3px, 色は Primary または #bdbdbd |
| **アイコン** | 32～64px, 単色（フォント色または #1976d2） |
| **余白** | セル間隔は最小 16px (md スペーシング) |
