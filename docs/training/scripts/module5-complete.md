# 研修動画スクリプト - Module 5: サイバーセキュリティ編（続き）

## 動画 5-2: セキュリティ対策の基本【12分】

- シーン1（0:00～1:00）：セキュリティ対策の 3 層防御モデル（人・プロセス・技術）
- シーン2（1:00～3:00）：第 1 層「人」の対策：従業員教育とフィッシング詐欺対応
- シーン3（3:00～5:00）：第 2 層「プロセス」の対策：アクセス管理、パスワード管理、バックアップ
- シーン4（5:00～7:00）：第 3 層「技術」の対策：ファイアウォール、ウイルス対策、暗号化
- シーン5（7:00～9:00）：保険代理店向けの「最小限セキュリティチェックリスト」
- シーン6（9:00～12:00）：投資対効果分析（セキュリティ投資 vs リスク軽減効果）

## 動画 5-3: 従業員教育と社内体制【11分】

- シーン1（0:00～1:00）：セキュリティの対策は「人」が 70%（根拠：NIST 調査）
- シーン2（1:00～3:00）：従業員教育の内容（フィッシング詐欺、社会工学、情報削除ルール）
- シーン3（3:00～5:00）：継続教育プログラムの実装（月 1 回のセキュリティニュースレター、四半期ごと教育）
- シーン4（5:00～7:00）：セキュリティ推進室の設置と責任体制
- シーン5（7:00～9:00）：インターンシップやアルバイトへの教育方法
- シーン6（9:00～11:00）：実例：従業員教育で詐欺メール対応率を 90% から 0% に改善した中小企業

## 動画 5-4: インシデント対応計画【12分】

- シーン1（0:00～1:00）：インシデントは「起きる」前提で計画を立てる
- シーン2（1:00～3:00）：インシデント対応の 5 つのステップ（検知 → 初期対応 → 調査 → 復旧 → 報告）
- シーン3（3:00～5:00）：インシデント対応チームの組成と役割分担
- シーン4（5:00～7:00）：顧客への報告義務と対外発表ガイドライン
- シーン5（7:00～9:00）：ランサムウェア被害に特化した対応手順
- シーン6（9:00～12:00）：実例：ランサムウェア攻撃への対応（4 時間で基幹システム復旧）

## 動画 5-5: 保険代理店向けセキュリティ標準【13分】

- シーン1（0:00～1:00）：保険代理店に求められるセキュリティ基準（金融庁ガイドライン）
- シーン2（1:00～3:00）：FISC（金融情報システムセンター）安全対策基準の概要
- シーン3（3:00～5:00）：個人情報保護法の要件（本人同意、安全対策、漏洩報告）
- シーン4（5:00～8:00）：保険業界独自のセキュリティ基準と監査のポイント
- シーン5（8:00～10:00）：金融監動官による抜き打ち監査の事例と対応方法
- シーン6（10:00～13:00）：代理店向けセキュリティチェックリスト（50 項目）と自己評価方法

## 動画 5-6: 継続的改善とコンプライアンス【12分】

- シーン1（0:00～1:00）：セキュリティは「一度きり」ではなく「継続的改善」
- シーン2（1:00～3:00）：セキュリティ監査の実施方法（内部監査 vs 外部監査）
- シーン3（3:00～5:00）：脅威インテリジェンス情報の収集と対応（新型マルウェアなど）
- シーン4（5:00～7:00）：法規制の改正への追従（個人情報保護法改正など）
- シーン5（7:00～9:00）：セキュリティ文化の醸成（経営層への重要性説得、従業員のセキュリティ意識向上）
- シーン6（9:00～12:00）：保険代理店の『セキュリティ × CSV』戦略（セキュリティサービス販売、セキュリティ診断サービス）
