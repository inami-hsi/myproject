# 研修動画スクリプト - Module 3: CSR基礎編（続き）

## 動画 3-2: CSR の国際基準（ISO 26000 と SDGs）【11分】

- シーン1（0:00～1:00）：CSR の国際基準。ISO 26000 と SDGs の関係性
- シーン2（1:00～3:30）：ISO 26000 の 7 つの中核課題（組織統治、人権、労働慣行、環境、公正、消費者、コミュニティ）
- シーン3（3:30～6:00）：SDGs（国連 2030 年目標）との連携。保険業界での対応事例
- シーン4（6:00～8:00）：企業が CSR を測定する 4 つの方法（報告書、ESG スコア、認証、ステークホルダー対話）
- シーン5（8:00～11:00）：保険代理店が ISO 26000 と SDGs を顧客支援に活用する方法

## 動画 3-3: CSR と企業の持続可能性【11分】

- シーン1（0:00～1:00）：CSR は「チャリティ」ではなく「持続可能性を守る経営戦略」
- シーン2（1:00～3:00）：企業の 3 つの資本（有形資本・人的資本・社会関係資本）とバランス
- シーン3（3:00～6:00）：CSR が 3 つの資本を増加させるメカニズム。3 つの実例（労働慣行、環境、顧客課題）
- シーン4（6:00～8:30）：CSR を無視した場合のリスク（負の循環）
- シーン5（8:30～11:00）：保険代理店の持続可能性と CSR の役割（BCP 支援、教育、セキュリティ強化）

## 動画 3-4: CSR の測定・報告・評価【12分】

- シーン1（0:00～1:00）：CSR の測定と報告が必要な理由
- シーン2（1:00～3:00）：測定できないと生じる 4 つの問題（改善不明確、ROI 不明確、信頼喪失、規制非対応）
- シーン3（3:00～6:00）：CSR 報告書の 4 つのステップ（課題特定、目標設定、実施、測定）
- シーン4（6:00～9:00）：保険代理店の CSR 報告書実装例（防災セミナー、BCP 支援、セキュリティ対応）
- シーン5（9:00～12:00）：継続的改善と次ステップ。Module 4 への接続（CSV 実践へ）
